-Сихроната функција се извршува линија по линија, прво треба д азврши една опеерација па да почне следната, се извршува по редоследн како што е напишан кодот ;
асихроната извршува во позадина н, не чека да заврши една операција туку параллелно се одвиваат и не го блокира евент loop, користиме async await
